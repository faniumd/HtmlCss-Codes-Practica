DESCRIPCIÓN DEL PROYECTO:
Web Interactiva sobre el anime de one punch man, recolectando
algunos de los héroes y villanos mas importantes del anime
TECNOLOGÍA USADAS:
HTML5 (Estructura semántica).
CSS3 (Flexbox para el Layout, Animaciones Keyframes)
Características Especiales:
Sidebar Fijo: Navegación persistente a la izquierda.
Carrusel Automático: Galería de personajes con efecto de enfoque (scale) y pausa al pasar el mouse.
Secciones de codigo demostrativas de Grid Y flexbox como ejemplo.
Instalación: “Clona la repo en Github y abrir el index.html. En tu editor de código favorito.”
Link: del Repositorio en Github: https://github.com/faniumd/HtmlCss-Codes-Practica.git
